# QuicksApp
WhatsApp has over two billion users worldwide, making it the most popular chat app for everyone. However, users encounter a significant issue when attempting to text an unknown person without having to save the number. In other words, users cannot text an unsaved number using the texting app. To do so, users must save the number to their phone and then text them using WhatsApp.

Quick-sApp. An android application built on the WhatsApp API that allows you to send WhatsApp messages without saving the recipient’s phone number.
